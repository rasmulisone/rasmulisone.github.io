Dice Roller

A simple dice roller application with unlimited number of dice and on Android and web a separate two-dice mode.
Made with: Sketchware (Android version) / Python 3 (Python 3 version) / HTML, JavaScript (web version)
Available platforms: Android, Python 3 (packages required: random), web
Tested platforms: Android 7.0 Nougat, QPython3 on Android 7.0 Nougat, Python 3.5 on Windows 10, Android 4.4 KitKat, Android 8.1 Oreo, Python 3.6 on Windows 10, Mozilla Firefox on Windows 10
Latest version: 1.4.1 (Android) / 1.1 (Python) / 1.1.1 (web)
