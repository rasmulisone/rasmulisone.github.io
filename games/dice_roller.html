Dice Roller

A dice roller application with unlimited number of dice with the result of each individual die (in the Android version only up to 20) and the total result. The Python version also includes an average result mode.
Made with: Sketchware (Android version) / Python 3 (Python version) / HTML, CSS, JavaScript (web version)
