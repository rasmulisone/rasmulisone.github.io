RandomPicker

A simple application that picks a random one of the chosen options. On Android and web it can also pick numbers between two specified numbers and on web numbers with a specified amount of digits.
Made with: Python 3 (Python version) / Sketchware (Android version) / HTML, CSS, JavaScript (web version)
