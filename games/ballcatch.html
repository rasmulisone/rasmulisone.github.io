BallCatch

A simple game with balls falling from top and you have to catch them. Game manual >>>
Made with: Unity 5.6.0 - 5.6.6, C#
Available platforms:
PlatformTested
Android (4.4+)7.0 Nougat
8.1 Oreo
9.0 Pie
Windows10
WebMozilla Firefox 62 on Windows
Latest version: 1.4.3
What's new:
1.4.3
  • Added keyboard shortcuts for the powerups
  • The mode can no longer be changed while the game is on
  • Changed the updater background color
1.4.2
  • Bugfix: if you lose while the "half speed" powerup is on, the counter will continue below zero seconds after you start a new game
  • The "half speed" counter now shows "OFF" at start instead of "0 s"
1.4.1
  • Bugfix: when you start a new game, the balls immediately fall at the speed they were falling when you lost if you haven't restarted the game
  • Added game manual
1.4 (Windows, Android)
  • Added powerup mode
  • Added "Checking for updates..." text
1.3.1 (Windows, Android)
  • Highscore is now checked even if you exit through the pause menu
  • Improved the position of the pause menu
  • Added version information to the update checker
  • Windows: the pause menu doesn't open anymore (by pressing esc) if the update checker is visible
1.3
  • Added pause menu
  • Windows, Android: added automatic checking for updates when starting the app
  • Windows: added downloading as an .exe installer
1.2.2
  • Bugfix: the Rasmulisone Games logo is stretched
  • Improved the graphics quality settings
  • Windows: made the resolution dialog be skipped if you don't press Alt while starting the app
1.2.1
  • Made the buttons more transparent to more easily see balls through them
  • Added version text
  • Added support for more aspect ratios (tested only with 16:9 and 18:9)
  • Updated the Rasmulisone Games logo
  • Web: removed unnecessary quit button
  • Web: fixed the aspect ratio
1.2
  • Each ball now spawns in a random color
  • Added quit button
  • Switched Unity version from 5.6.1 to 5.6.6
  • Android: reduced app size by disabling Android TV and x86 support
  • Android: changed minimum SDK version from 16 (4.1 Jelly Bean) to 19 (4.4 KitKat)
1.1
  • The catching platform moves faster
  • If you now move the catching platform over the screen edge, it will move to the other screen edge
Known bugs / missing features:
  • If you lose while the "half speed" powerup is on, the counter will continue below zero seconds after you start a new game (1.4 - 1.4.1, fixed in 1.4.2)
  • When you start a new game, the balls immediately fall at the speed they were falling when you lost if you haven't restarted the game (1.4, fixed in 1.4.1)
  • The Rasmulisone Games logo is stretched (1.2.1, fixed in 1.2.2)
Installing:
  • Android: Download and install the apk
  • Windows (installer): Download and start the .exe file
  • Windows (zip): Download, unzip and start the .exe file
