BallCatch

A simple game with balls falling from top and you have to catch them.
Made with: Unity 5.6.0 - 5.6.6, C#
Available platforms: Android (4.4 KitKat and up), Windows, web (Untested Linux version available here)
Tested platforms: Android 7.0 Nougat, Windows 10, Android 8.1 Oreo, Mozilla Firefox on Windows 10
Latest version: 1.4 (Windows, Android) / 1.3 (web)
What's new:
1.4 (Windows, Android)
  • Added powerup mode
  • Added "Checking for updates..." text
1.3.1 (Windows, Android)
  • Highscore is now checked even if you exit through the pause menu
  • Improved the position of the pause menu
  • Added version information to the update checker
  • Windows: the pause menu doesn't open anymore (by pressing esc) if the update checker is visible
1.3
  • Added pause menu
  • Windows, Android: added automatic checking for updates when starting the app
  • Windows: added downloading as an .exe installer
1.2.2
  • Fixed the Rasmulisone Games logo being stretched
  • Improved the graphics quality settings
  • Windows: made the resolution dialog be skipped if you don't press Alt while starting the app
1.2.1
  • Made the buttons more transparent to more easily see balls through them
  • Added version text
  • Added support for more aspect ratios (tested only with 16:9 and 18:9)
  • Updated the Rasmulisone Games logo
  • Web: removed unnecessary quit button
  • Web: fixed the aspect ratio
1.2
  • Each ball now spawns in a random color
  • Added quit button
  • Android: reduced app size by disabling Android TV and x86 support
  • Android: changed minimum SDK version from 16 (4.1 Jelly Bean) to 19 (4.4 KitKat)
1.1
  • The catching platform moves faster
  • If you now move the catching platform over the screen edge, it will move to the other screen edge
Installing:
  • Android: Download and install the apk
  • Windows (installer): Download and start the .exe file
  • Windows (zip): Download, unzip and start the .exe file
