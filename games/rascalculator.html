rasCalculator

A simple Python 3 command line calculator with +, -, *, /, exp, sqrt, %of and -%.
Made with: Python 3
Available platforms: Python 3 (packages required: math)
Tested platforms: QPython3 on Android 7.0 Nougat
Latest version: 1.0.1
What's new:
1.0.1
  • Bugfix: When substracting, the original numbers were shown before the result (a debug feature accidentally present in the release version)
  • If you type something else than a supported command, the list that shows up shows all the supported commands, not just some as it used to
  • Added "exit" to the visible list of commands at the start
