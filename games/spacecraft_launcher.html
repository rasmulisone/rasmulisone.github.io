Spacecraft Launcher

An automatic 3D spacecraft launcher app with multiple views and a few options to customize the launch. MECO (main engine cutoff) should be at around T+ 0:4:22 (version 0.3 beta and up).
Made with: Unity 5.6.1, C#
Available platforms:
PlatformTested
Windows10
Latest version: 0.3 beta
What's new:
0.3 beta
  • Changed rocket model
  • Added second stage of rocket and first stage separation
  • Rocket's mass now decreases as fuel is burned
  • Added atmospheric drag that decreases as the rocket goes up to the skies
  • Added acceleration per second meter
  • Improved smoke effects
  • Improved rocket engine flame effect
  • UI now scales to resolution
  • Changed terrain to be smaller
  • Removed some trees
0.2 beta
  • Added option to toggle fog
  • Added option to start paused
  • Added option to disable smoke on launch
  • Added option to start at T- 10 seconds or 1 minute
0.1.1 alpha
  • Added pausing
  • Added exiting from simulation
  • Added controls screen
Known bugs / missing features:
  • Not all smoke is disabled when smoke disabled
  • No manual controlling mode
  • No more spacecraft than one
  • Unrealistic environment
  • No more environments than one
  • No possibility to set a flight path for the rocket
  • No clouds
  • No possibility for the rocket to explode, crash etc.
  • No wind, rain etc.
  • No free camera
Installing: Download, unzip and start the .exe file.
